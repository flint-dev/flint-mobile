# Getting Started

---

## Clone the repository and install dependencies

```
git clone https://github.com/flint-dev/flint-mobile.git
cd flint-mobile
yarn
```

---

## Running the app

```
run yarn start
open Expo Go and scan the generated QR code to open the app on mobile
```
